Project Regulus API Documentation

Recent Grades

http://emoosx.me/regulus/api/grades/recentGrades/sid/{sid}/password/{password}

Results

  • module_code
  • problem
  • grade
  • individual_comment
  • team_comment
  • attendance

Recent UT Grades

http://emoosx.me/regulus/api/grades/recentUTGrades/sid/{sid}/password/{password}

Results

  • module_code
  • ut_no
  • ut_grade

  • ce_nondiploma




RJ Question of the Day

http://emoosx.me/regulus/api/dailyProblem/rjquestion/sid/{sid}/password/{password}

Results

  • problem_name
  • rj_question




Don't forget to URLEncode the password since people tend to use special characters such as "#" in their passwords

The default in JSON. If you prefer XML, add format/xml at the end of the URL.
