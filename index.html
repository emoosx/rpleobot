RPLeoBot

This is my attempt to ease out repetitive LEO tasks targeting mainly at RP students.
Deployed on Google App Engine.
Following features are being supported.

  • /grades => to check out the recently published grades & published UT grades
  • /rj => to check out the RJ question of the day (includes submission status)
  • /timetable => to check out the timetable
  • /ce => view CE point status
  • /gpa => view cumulative GPA
  • /me => view personal particulars

Getting Started

Simply just add rpleobot@appspot.com in your google talk. Type anything. A usage manual will pop up. Have fun!
